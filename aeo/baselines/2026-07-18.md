## Patterns

1. **Reddit dominates.** 17/50 results are Reddit threads. IE has r/invisibleexit but it doesn't 
   rank for any query — likely because the subreddit has minimal activity. AEO fix: seed the
   subreddit with 10+ high-value posts, then answer genuinely in the threads that DO rank.

2. **"Freedom number" is unowned.** The query returns FI calculators from Fidelity/Reddit/LinkedIn,
   not IE's freedom number calculator. This is IE's strongest owned-concept opportunity — the
   term was coined by IE. Fix: get the freedom number calculator linked from third-party sites
   and write guest posts that use/define the term with attribution.

3. **"Anonymous LLC" belongs to law firms.** IE's blog posts on anonymous LLCs, entity separation,
   and state guides don't rank. This content exists (53 blog posts) but has no authority.
   Fix: build backlinks to the state guide pages (they're the most linkable SEO asset).

4. **Competitors win by owning their branded queries.** StarterStory and SideHustleNation only
   appear for query #7 (branded alternatives search). They don't rank for generic category
   queries either — the category is fragmented across Reddit/UGC.

5. **No listicles cite any tool in this space.** None of the 50 results are "best X" listicles
   from editorial sites. This means the category doesn't have established tier-1 editorial
   sources — which is an OPPORTUNITY. If IE gets onto the sites that AI pulls from before the
   category matures, early-mover advantage compounds.
