MCP Server

Invisible Exit — MCP Server

Complete guide to digital privacy and disappearing from the internet.

One endpoint. Stateless HTTP transport. No auth required. Paste it into any MCP client:

https://invisibleexit.com/api/mcp

Install in Claude Desktop

Add to claude_desktop_config.json:

{
  "mcpServers": {
    "invisibleexit": {
      "command": "npx",
      "args": ["mcp-remote", "https://invisibleexit.com/api/mcp"]
    }
  }
}

Install in Cursor

Add to .cursor/mcp.json:

{
  "mcpServers": {
    "invisibleexit": {
      "url": "https://invisibleexit.com/api/mcp"
    }
  }
}

Inspect

npx @modelcontextprotocol/inspector https://invisibleexit.com/api/mcp

Available Tools (3)

  • search_privacy_guides — Search privacy guides by topic.
  • get_privacy_guide — Get a detailed step-by-step privacy guide.
  • check_data_brokers — List major data brokers with opt-out links.
